Arrange the cards so that in the top 25 circles, all the cards in the same row, column, or diagonal have the same character in common. (Remember that diagonals wrap around.)

Start by choosing any character, and placing all the cards with that character in the bottom six circles. (These six circles form the line at infinity in the projective plane.)

If you need some help, you can click on the "Show Lines" button, which highlights the lines that have been correctly formed.
